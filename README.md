# JGPhotoBrowser

 图片大图浏览查看工具，支持GIF图片。改写自 Sunnyyoung/MJPhotoBrowser (https://github.com/Sunnyyoung/MJPhotoBrowser)
 
 功能包括：
 1、浏览图片大图，支持手势缩放
 2、支持图片保存、支持GIF图片保存
 3、增加图片介绍展示

## 来源

https://github.com/Sunnyyoung/MJPhotoBrowser
